## Prerequisites

To run this project, you will need a Linux/Unix-based environment. This includes:
* A Linux Virtual Machine or native Linux OS (I used Oracle VirtualBox - Linux Mint)
* macOS terminal
* Windows Subsystem for Linux (WSL) or Git Bash on Windows

## How to Run
1) Clone this repository to your local machine:
   
```git clone https://github.com/mirunq-png/sys-man-sim.git```

2) Navigate into the project directory:

```cd sys-man-sim```

3) Grant execute permissions to all scripts:

```chmod u+x *.sh```

4) Execute the main script:

```./main.sh```
